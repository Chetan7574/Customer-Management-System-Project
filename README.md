# Customer Management System Project
This capstone Project was build to analyse data and retrive the data cuts for resports.
• Designed and implemented a relational database with 2,000+ customer records and 5,000+ order transactions to efficiently manage retail sales and customer information.

• Wrote advanced SQL queries using joins, subqueries, aggregations, and window functions to analyze monthly sales trends, top 10% high-spending customers, and 20+ product performance indicators.
